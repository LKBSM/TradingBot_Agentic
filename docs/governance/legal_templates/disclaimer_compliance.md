Les marchés financiers comportent un **risque de perte en capital**.

>
> **Autonomie de l'utilisateur.** L'utilisateur prend ses décisions d'investissement en toute autonomie, sur la base de sa propre analyse et après évaluation de sa situation patrimoniale, de ses objectifs et de sa tolérance au risque. Il est encouragé à consulter un conseiller financier indépendant agréé avant toute décision d'investissement.
>
> **Phase Early Access.** Le Service est en phase d'accès anticipé. Les fonctionnalités, les performances et les tarifs peuvent évoluer. Aucune obligation de continuité de service n'est garantie pendant cette phase.

## 🚫 Vocabulaire INTERDIT (ne JAMAIS utiliser)

À éliminer partout : marketing, narratives LLM, Telegram, emails, API docs, chatbot, screenshots.

| ❌ Interdit FR | ❌ Interdit EN | ✅ Remplacer par |
|---|---|---|
| signal de trading | trading signal | lecture algorithmique / analyse |
| acheter / vendre | buy / sell | setup haussier / baissier |
| BUY / SELL | BUY / SELL | BULLISH SETUP / BEARISH SETUP |
| garanti / garantie | guaranteed | calibré / vérifié sur historique |
| profit X % | X % profit | profit factor historique de X |
| gagnez | earn / win | comprenez / analysez |
| recommandation | recommendation | analyse / lecture |
| conseil | advice | éclairage / contexte |
| opportunité | opportunity | setup / configuration |
| ça va monter / baisser | will go up / down | conditions haussières / baissières |
| stop-loss / take-profit | stop-loss / take-profit | niveau d'invalidation structurelle |
| target / cible | target | niveau de liquidité supérieur |
