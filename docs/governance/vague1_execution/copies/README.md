## Règle d'or

**Vocabulaire interdit** (à ne JAMAIS écrire dans les copies — réf. `legal_templates/disclaimer_compliance.md`) :
- signal de trading, trading signal
- acheter / vendre, buy / sell, BUY / SELL
- garanti, guaranteed
- profit X %, X % profit
- gagnez, earn, win
- recommandation, recommendation
- conseil, advice
- opportunité, opportunity
- ça va monter / baisser, will go up / down
- stop-loss / take-profit prescriptif
