# MISSION_ACK — Renaming Smart Sentinel AI

**Date** : 2026-05-26
**Branche** : institutional-overhaul
**Sources lues** : `docs/value/client_information_explained.txt`, `docs/value/client_relevance_review.md`, `docs/value/best_product_concept.md`, `docs/governance/decisions/2026-05-26_5_political_locks.md`, `docs/governance/decision_gate_review_v2.md` (skim).

## Compréhension produit en 5 lignes

1. **Quoi** : indicateur de marché conversationnel (Or XAU + FX EUR) couplé à un chatbot pédagogique, livré sur 3 surfaces (Telegram compact, webapp progressive, API B2B JSON).
2. **Promesse** : transformer le bruit du marché en lecture claire — track record honnête en hero (PF historique + IC bootstrap sur N setups), conviction calibrée isotoniquement, marge d'erreur conformelle assumée.
3. **Posture** : éducative et descriptive, jamais prescriptive. Pas de buy/sell, pas de SL/TP, pas de R:R. Lecture haussière / baissière / illisible — le trader décide.
4. **Cible** : trader retail B2C grand public engagé, FR-first puis EN, 25-55 ans, du débutant motivé au power-user SMC. Marché adressable FR ~15-30k individus.
5. **Anti-positionnement** : pas un service de signaux (compliance UE 2024/2811 + MiFID finfluencer mars 2026), pas un bot de trading, pas une promesse de gain (`edge_claim = False` codifié, valable tant que PF live <1.20 sur 12 mois).

## Pourquoi rebaptiser

Trois raisons identifiées par le brief, toutes valides après lecture des sources :
- **"Smart"** = préfixe épuisé en tech (Smart TV, Smart Home, SmartThings) → marqueur de paresse marketing.
- **"Sentinel"** = collision frontale avec SentinelOne (unicorn cybersécurité cotée NYSE, market cap >$5B), donc territoire cyber occupé. Aussi : connotation prescriptive "garde armée" qui colle mal à la posture éducative.
- **"Smart Sentinel AI"** ne raconte rien de ce que le produit fait : ni indicateur, ni conversation, ni honnêteté statistique, ni lecture pédagogique.

## Contraintes structurantes pour le nom

- **Compliance** : ne doit JAMAIS évoquer signaux, trades, BUY/SELL, profit garanti. Vocabulaire officiel = "analyses", "lectures algorithmiques" (lock politique signé 2026-05-26).
- **Posture multi-vues** : nom doit pouvoir porter les 3 modes FOCUS / CO-PILOT / EXPERT sans biais (donc pas trop "simple" qui plombe EXPERT, pas trop "institutionnel" qui plombe FOCUS).
- **Wedge GTM FR-first** (eval_25) : prononçable en français au premier essai, sans épellation.
- **Tier INSTITUTIONAL $1990** (eval_27, lock 2026-05-26) : nom doit tenir face à un broker / family office sans paraître crypto-bro.
